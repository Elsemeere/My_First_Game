# Maze Golf

A simple 2D maze mini-golf game built with plain HTML, CSS, and JavaScript. No frameworks, no build tools — just open the file and play.

## How to play

1. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
2. Click and drag away from the ball to aim.
3. Release to shoot. The further you drag, the harder the shot.
4. Get the ball into the hole (marked with a red flag) in as few shots as possible.
5. Press **Reset** to restart the level at any time.

## Controls

| Action | Input |
|--------|-------|
| Aim | Click and hold on the canvas, drag away from the ball |
| Shoot | Release the mouse button |
| Reset level | Click the Reset button |

Touch input (tablet/phone) is also supported.

## Running locally

No server needed. Just double-click `index.html` or drag it into a browser tab.
